EXERCÍCIOS DE LÓGICA EM PYTHON
Este repositório foi criado para armazenar meus estudos e exercícios práticos desenvolvidos na linguagem Python. Cada pasta contém um script focado em um conceito específico de programação, desde interações básicas até estruturas de repetição e tratamento de erros.

ESTRUTURA DO REPOSITÓRIO
O repositório está organizado em pastas numeradas para facilitar a navegação:

• 01-ola-personalizado: Script inicial para prática de entrada de dados e formatação de strings.

• 02-calculadora-simples: Implementação de uma calculadora com menu interativo e estrutura match-case.

• 03-par-ou-impar: Exercício focado em operadores aritméticos (módulo) e estruturas condicionais.

• 04-tabuada: Gerador de tabuadas completas utilizando laços de repetição (for) e lógica de operações inversas.

• 05-media-notas: Sistema simples para cálculo de média aritmética e verificação de aprovação escolar.

TECNOLOGIAS E CONCEITOS APLICADOS
• Linguagem: Python 3

• Entrada e Saída: Funções input() e print().

• Estruturas Condicionais: if, else e match-case.

• Laços de Repetição: while para menus contínuos e for para iterações numéricas.

• Tratamento de Exceções: Uso de blocos try/except para lidar com entradas inválidas.

• Formatação de Dados: Uso de f-strings para alinhamento e precisão decimal.

COMO UTILIZAR ESTE REPOSITÓRIO
Para executar qualquer um dos exercícios localmente, siga estes passos:

1. Clone o repositório:

git clone https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git

2. Navegue até a pasta do exercício desejado:

cd 04-tabuada

03. Execute o script:

python main.py
